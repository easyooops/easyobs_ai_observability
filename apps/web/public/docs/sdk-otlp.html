OTLP/HTTP (Any Language)

OTLP/HTTP Integration

EasyObs accepts standard OTLP/HTTP traces from any language that has an OpenTelemetry SDK ??Java, Go, .NET, Rust, Ruby, PHP, and more.

Endpoints

PathDescription
/v1/tracesStandard OTLP endpoint (use with any OTel SDK)
/otlp/v1/tracesLegacy EasyObs path (Python SDK uses this)

Authentication

Authorization: Bearer <your-ingest-token>

Tokens are minted in Setup > Organizations > Services.

cURL Example

curl -X POST http://localhost:8787/v1/traces \
  -H "Authorization: Bearer eobs_your_token" \
  -H "Content-Type: application/json" \
  --data @sample_ingest.json

JSON Body Format

{
  "resourceSpans": [{
    "resource": {
      "attributes": [
        {"key": "service.name", "value": {"stringValue": "my-agent"}}
      ]
    },
    "scopeSpans": [{
      "spans": [{
        "traceId":  "5b8aa5a2d2c872e8321cf37308d69df2",
        "spanId":   "051581bf3cb55c13",
        "name":     "agent.turn",
        "kind":     1,
        "startTimeUnixNano": "1730000000000000000",
        "endTimeUnixNano":   "1730000000750000000",
        "status":   {"code": 1},
        "attributes": [
          {"key": "o.kind",    "value": {"stringValue": "agent"}},
          {"key": "o.q",       "value": {"stringValue": "summarise yesterday"}},
          {"key": "o.r",       "value": {"stringValue": "Here is..."}},

  }]
}
