빠른 시작

3분 안에 EasyObs를 로컬에서 실행하고 첫 트레이스를 전송해 봅니다.

사전 요구사항

도구최소 버전설명
Python3.11+FastAPI 서버용
Node.js20+Next.js 웹 콘솔용
npm9+Node.js에 함께 포함됨
uv (선택)0.1+가상 환경을 자동 관리합니다 (권장)

Step 1: 프로젝트로 이동

# 저장소 클론 및 디렉터리 진입
git clone <repository-url>
cd easyobs_ai_observability

Step 2: 환경 파일 생성

# 샘플 .env 파일 복사
cp .env.sample .env          # Linux/macOS
Copy-Item .env.sample .env   # Windows PowerShell

기본 설정만으로도 바로 동작합니다. 데모 데이터를 자동 생성하려면 EASYOBS_SEED_MOCK_DATA=true를 설정하세요.

Step 3: 서버 시작

Windows (PowerShell)

.\scripts\run-dev.ps1

Linux / macOS

chmod +x scripts/run-dev.sh
./scripts/run-dev.sh

이 스크립트는 다음을 자동으로 수행합니다.

  1. Python 가상 환경 생성 및 의존성 설치
  2. npm 패키지 설치 (apps/web)
  3. FastAPI 서버를 백그라운드로 실행 (포트 8787)
  4. Next.js 개발 서버를 포그라운드로 실행 (포트 3000)

Step 4: 콘솔 접속

서비스URL설명
Web Consolehttp://localhost:3000메인 대시보드
API Serverhttp://127.0.0.1:8787REST API 엔드포인트
OpenAPI Docshttp://127.0.0.1:8787/docsSwagger UI (API 탐색기)
Health Checkhttp://127.0.0.1:8787/healthz서버 상태 확인
💡 첫 사용자 자동 승급: 가장 먼저 가입한 사용자는 자동으로 Super Admin(SA)이 되며, 기본 administrator 조직에 PO로 추가됩니다.

Step 5: 첫 트레이스 전송

에이전트 SDK 설치:

pip install -e ".[agent]"

최소 통합 코드:

from easyobs_agent import init, traced

init(
    "http://127.0.0.1:8787",
    token="eobs_...",       # Organizations > Services 메뉴에서 발급
    service="my-agent",
)

@traced("agent.turn")
def answer(query: str) -> str:
    return llm.invoke(query).content

Step 6: 데모 데이터 살펴보기

EASYOBS_SEED_MOCK_DATA=true가 설정되면 서버 첫 부팅 시 합성 트레이스를 자동으로 생성합니다.

  • 기본 1,000건의 합성 트레이스
  • 10일 시간 범위에 분산
  • Overview, Tracing, Interactions 페이지에서 즉시 탐색 가능

서버 옵션

포트 변경

# Windows
.\scripts\run-dev.ps1 -ApiPort 9000 -WebPort 4000

# Linux/macOS
API_PORT=9000 WEB_PORT=4000 ./scripts/run-dev.sh

의존성 설치 건너뛰기 (재시작)

.\scripts\run-dev.ps1 -SkipInstall      # Windows
./scripts/run-dev.sh --skip-install     # Linux/macOS

데이터 초기화

.\.venv\Scripts\easyobs.exe reset-data --force --yes   # Windows
./.venv/bin/easyobs reset-data --force --yes           # Linux/macOS
