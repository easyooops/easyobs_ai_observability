API 레퍼런스

EasyObs는 FastAPI로 구축된 RESTful API를 제공합니다. 인터랙티브 문서는 /docs(Swagger UI)에서 확인할 수 있습니다.

Interactive API Explorer — http://127.0.0.1:8787/docs에서 try-it-out을 지원하는 라이브 Swagger UI를 사용할 수 있습니다. Open Swagger UI

인증

모든 API 요청은 Bearer 토큰을 통한 인증이 필요합니다.

Authorization: Bearer <token>

두 가지 토큰 타입이 존재합니다.

Type용도발급 방법
Session JWT웹 콘솔 / 관리 API.POST /v1/auth/signin
Ingest TokenSDK에서 OTLP 트레이스 수집.Organizations > Services

Base URL

http://127.0.0.1:8787    (로컬 개발)
https://easyobs.your.co  (리버스 프록시 뒤의 운영)

엔드포인트 그룹

Health & System

MethodPath설명
GET/healthz헬스 체크.

Authentication

MethodPath설명
POST/v1/auth/signup새 사용자 등록.
POST/v1/auth/signin로그인 후 JWT 발급.
GET/v1/auth/me현재 사용자 정보 조회.

Trace Ingestion

MethodPath설명
POST/v1/tracesOTLP/HTTP 표준 ingest.
POST/otlp/v1/traces레거시 EasyObs ingest 경로.

Traces & Analytics

MethodPath설명
GET/v1/traces트레이스 목록 (페이지네이션·필터).
GET/v1/traces/{id}span을 포함한 트레이스 상세.
GET/v1/sessions세션 목록.
GET/v1/analytics/overview대시보드 KPI와 시계열.

Organizations

MethodPath설명
GET/v1/organizations조직 목록.
POST/v1/organizations조직 생성.
GET/v1/organizations/{id}조직 상세.

POST/v1/organizations/{id}/servicesservice 생성 + 토큰 발급.
GET/v1/organizations/{id}/members멤버 목록.

Evaluations (Quality)

MethodPath설명
GET/v1/evaluations/profiles프로파일 목록.
POST/v1/evaluations/profiles프로파일 생성.
GET/v1/evaluations/runs실행 목록.
POST/v1/evaluations/runs새 실행 시작.
GET/v1/evaluations/runs/{id}실행 상세 + 결과.
GET/v1/evaluations/judges저지 모델 목록.
POST/v1/evaluations/judges저지 모델 등록.
GET/v1/evaluations/judge-prompts버전 관리되는 평가 프롬프트 목록 (선택: ?dimension_id 필터).
POST/v1/evaluations/judge-prompts새 프롬프트 버전 생성 (자동 활성화).
POST/v1/evaluations/judge-prompts/{id}:activate특정 프롬프트 버전 활성화.
GET/v1/evaluations/judge-dimensions평가 디멘션 카탈로그 조회.
GET/v1/evaluations/golden-sets골든 셋 목록.
POST/v1/evaluations/golden-sets골든 셋 생성.
POST/v1/evaluations/golden-sets/{id}/upload항목 업로드 (CSV/JSONL/xlsx).
GET/v1/evaluations/improvementsimprovement 제안 목록.
PATCH/v1/evaluations/improvements/{id}상태 업데이트 (triage/apply/reject).

공통 응답 코드

Code의미
200성공.
201생성됨.
400유효성 검사 오류 (요청 바디를 확인).
401토큰 누락 또는 잘못됨.
403권한 부족.
404리소스 없음.
429rate limit 도달.
500내부 서버 오류.

Rate Limiting

ingest 토큰 기준 기본 한도:

  • 트레이스 ingest: 1000 spans/second
  • 관리 API: 100 requests/minute
