환경 설정

모든 환경 변수는 선택 사항입니다. 기본값은 src/easyobs/settings.py에 정의되어 있습니다.

설정 방법: .env 파일에 추가하거나, 컨테이너 또는 서버리스 런타임에 실제 환경 변수로 주입할 수 있습니다.

스토리지

변수기본값설명
EASYOBS_DATA_DIR./dataSQLite 카탈로그, JWT 시크릿, 트레이스 블롭이 저장되는 디렉터리. easyobs reset-data 실행 시 비워집니다.
EASYOBS_DATABASE_URL(자동)비워두면 <DATA_DIR>/catalog.sqlite3이 사용됩니다. 프로덕션 예시: postgresql+asyncpg://user:pw@host:5432/easyobs

HTTP 서버

변수기본값설명
EASYOBS_API_HOST127.0.0.1바인딩 주소. 컨테이너 환경에서는 0.0.0.0을 사용하세요.
EASYOBS_API_PORT8787API + OTLP/HTTP 수집 포트
EASYOBS_CORS_ORIGINShttp://127.0.0.1:3000,...쉼표로 구분되는 브라우저 origin 화이트리스트

인증 / 세션

변수기본값설명
EASYOBS_JWT_SECRET(자동 생성)HS256 서명 키. 첫 실행 시 <DATA_DIR>/jwt.secret에 자동 생성됩니다. 프로덕션에서는 openssl rand -hex 32로 직접 생성한 값을 사용하세요.
EASYOBS_JWT_TTL_HOURS12JWT 만료 시간(시간 단위)

LLM 비용 가격 정책

변수기본값설명
EASYOBS_PRICING_SOURCEauto수집 시 o.price 보강에 사용할 가격 소스. auto는 tokencost → litellm → 내장 fallback 순으로 시도합니다.

로깅 / 운영 가시성

변수기본값설명
EASYOBS_LOG_LEVELINFO로그 레벨: DEBUG / INFO / WARNING / ERROR
EASYOBS_LOG_FORMATconsoleconsole: 사람이 읽기 쉬운 한 줄 형식. json: CloudWatch/Loki 호환 JSON 형식.
EASYOBS_LOG_FILE(없음)설정 시 stdout과 함께 해당 파일에도 로그를 남깁니다.
EASYOBS_LOG_REQUEST_BODYfalsetrue로 설정하면 요청 본문(최대 4KB)을 로그에 남깁니다. PII 유출 위험이 있어 기본은 비활성화입니다.

데모 / 모의 데이터

변수기본값설명
EASYOBS_SEED_MOCK_DATAfalsetrue로 설정하면 첫 부팅 시 데모 트레이스를 생성합니다.
EASYOBS_SEED_MOCK_TRACES100생성할 합성 트레이스 개수
EASYOBS_SEED_MOCK_WINDOW_HOURS24합성 트레이스를 분산시킬 시간 범위(시간 단위)
EASYOBS_SEED_MOCK_LIVEfalse서버 실행 중에도 합성 트레이스를 지속적으로 생성합니다.
EASYOBS_SEED_MOCK_LIVE_INTERVAL_SEC20라이브 합성 트레이스 생성 간격(초)

품질 (평가) 모듈

변수기본값설명
EASYOBS_EVAL_ENABLEDtrue마스터 스위치. false 시 평가 라우트와 UI가 모두 숨겨집니다.
EASYOBS_AUTO_RULE_ON_INGESTtrue트레이스 수집 시 룰 평가를 자동으로 트리거합니다.

알람 (임계값 알림)

변수기본값설명
EASYOBS_ALARM_ENABLEDtrue알람 모듈 마스터 스위치
EASYOBS_ALARM_EVAL_INTERVAL_SECONDS60알람 평가 주기(초)

프런트엔드 (웹 콘솔)

변수기본값설명
NEXT_PUBLIC_API_URLhttp://127.0.0.1:8787웹 콘솔이 호출할 API 베이스 URL

AWS Bedrock (Judge)

변수기본값설명
AWS_PROFILEdefaultboto3 세션 프로파일 (~/.aws/credentials 참조)

예시 .env 파일

# === 로컬 개발 ===
EASYOBS_DATA_DIR=./data
EASYOBS_API_HOST=127.0.0.1
EASYOBS_API_PORT=8787
EASYOBS_LOG_LEVEL=INFO
EASYOBS_LOG_FORMAT=console
EASYOBS_SEED_MOCK_DATA=true
EASYOBS_SEED_MOCK_TRACES=1000

# === 프로덕션 ===
# EASYOBS_DATABASE_URL=postgresql+asyncpg://user:pw@db:5432/easyobs
# EASYOBS_API_HOST=0.0.0.0
# EASYOBS_JWT_SECRET=your-strong-secret-here
# EASYOBS_LOG_FORMAT=json
# EASYOBS_SEED_MOCK_DATA=false
