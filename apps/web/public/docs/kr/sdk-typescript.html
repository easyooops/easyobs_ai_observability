TypeScript / JS SDK

TypeScript / JavaScript 통합

공식 OpenTelemetry JS SDK를 사용하여 Node.js 애플리케이션에서 EasyObs로 트레이스를 전송할 수 있습니다.

의존성 설치

npm install \
  @opentelemetry/api \
  @opentelemetry/sdk-node \
  @opentelemetry/exporter-trace-otlp-proto \
  @opentelemetry/resources \
  @opentelemetry/semantic-conventions

트레이싱 초기화

// tracing.ts — 진입 파일에서 가장 먼저 import 하세요
import { NodeSDK } from "@opentelemetry/sdk-node";
import { OTLPTraceExporter } from "@opentelemetry/exporter-trace-otlp-proto";
import { Resource } from "@opentelemetry/resources";
import { SemanticResourceAttributes } from "@opentelemetry/semantic-conventions";

export const sdk = new NodeSDK({
  resource: new Resource({
    [SemanticResourceAttributes.SERVICE_NAME]: "my-agent",
  }),
  traceExporter: new OTLPTraceExporter({
    url: "http://localhost:8787/v1/traces",
    headers: {
      Authorization: "Bearer eobs_your_token_here",
    },
  }),
});

sdk.start();
process.on("SIGTERM", () => sdk.shutdown().catch(() => {}));

스팬 생성

import { trace, SpanStatusCode } from "@opentelemetry/api";

const tracer = trace.getTracer("my-agent");

export function traced<T>(name: string, fn: () => Promise<T>): Promise<T> {
  return tracer.startActiveSpan(name, async (span) => {
    try {
      const out = await fn();
      span.setStatus({ code: SpanStatusCode.OK });
      return out;
    } catch (err) {
      span.recordException(err as Error);
      span.setStatus({ code: SpanStatusCode.ERROR });
      throw err;
    } finally {
      span.end();
    }
  });
}

// 사용 예시
export async function answer(q: string) {
  return traced("agent.turn", async () => {
    const span = trace.getActiveSpan()!;
    span.setAttribute("o.q", q);
    span.setAttribute("o.user", "alice");
    const reply = await llm.invoke(q);
    span.setAttribute("o.r", reply);
    return reply;
  });
}

중첩 스팬

export async function ragTurn(q: string) {
  return tracer.startActiveSpan("agent.turn", async (root) => {
    root.setAttribute("o.kind", "agent");
    root.setAttribute("o.q", q);

    const docs = await tracer.startActiveSpan("retrieve", async (sp) => {
      sp.setAttribute("o.kind", "retrieve");
      sp.setAttribute("o.target", "faq-index");
      try { return await vector.search(q); }
      finally { sp.end(); }
    });

    const reply = await tracer.startActiveSpan("generate", async (sp) => {
      sp.setAttribute("o.kind", "llm");
      sp.setAttribute("o.model", "gpt-4o-mini");
      try {
        const out = await llm.invoke({ q, docs });
        sp.setAttribute("o.tok.in", out.usage.input);
        sp.setAttribute("o.tok.out", out.usage.output);
        return out.text;
      } finally { sp.end(); }
    });

    root.setAttribute("o.r", reply);
    root.end();
    return reply;
  });
}

EasyObs 속성 네임스페이스

EasyObs 전용 속성은 o.* 프리픽스를 사용하세요.

속성설명
o.kind스팬 종류: agent, llm, retrieve, tool
o.q입력 질의/프롬프트
o.r출력 응답
o.modelLLM 모델 이름
o.tok.in / o.tok.out토큰 수 (입력/출력)
o.sess세션 ID
o.user사용자 식별자
OTLP/HTTP모든 언어를 위한 raw HTTP 통합 Python SDK네이티브 Python 클라이언트
