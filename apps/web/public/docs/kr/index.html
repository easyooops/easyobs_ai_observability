EasyObs 문서

LLM 에이전트를 위한 경량 관측 플랫폼입니다. OTLP/HTTP 기반 트레이싱, 멀티 테넌트 RBAC, 그리고 3-Way 평가 시스템을 제공합니다.

EasyObs란?

EasyObs는 LLM(Large Language Model) 에이전트 서비스를 위해 설계된 오픈소스 관측 플랫폼입니다. OpenTelemetry 프로토콜(OTLP/HTTP)을 기반으로 트레이스를 수집하며, 직관적인 웹 콘솔을 통해 모니터링·분석·평가 기능을 제공합니다.

주요 기능

🔍 트레이싱

모든 LLM 호출, 검색, 도구 사용을 스팬(span) 단위로 추적합니다. 지연 시간, 비용, 토큰 사용량을 실시간으로 모니터링합니다.

📊 인터랙션

여러 턴으로 이어지는 대화를 세션 단위로 묶고, 사용자별 참여도를 추적합니다. Sessions·Users 탭이 통합된 화면에서 한눈에 확인할 수 있습니다.

⚡ 품질 평가

Rule 기반, LLM-as-a-Judge, 휴먼 라벨링을 결합한 3-Way 평가를 제공합니다. 다중 저지(Judge) 합의와 비용 거버넌스까지 지원합니다.

🔗 SDK & 통합

Python SDK, TypeScript/JS, LangChain 콜백 핸들러를 제공하며, OTLP를 지원하는 모든 언어와 호환됩니다.

🏢 멀티 테넌트

Organization × Service의 2축 멀티 테넌시를 지원하며, SA/PO/DV 역할 기반 접근 제어(RBAC)를 제공합니다.

🚨 알람

임계값 기반 알림을 제공합니다. 에러율, 지연 시간, 비용 메트릭을 자동으로 모니터링하고 알림을 발송합니다.

빠르게 시작하기

3분 빠른 시작 — 로컬에서 EasyObs를 실행하고 첫 트레이스를 전송해 보세요. 빠른 시작 가이드 →
1
클론 및 환경 설정 
git clone <repository-url>
cd easyobs_ai_observability
cp .env.sample .env
2
서버 시작 (API + Web) 
./scripts/run-dev.ps1   # Windows
./scripts/run-dev.sh    # Linux/macOS
3
콘솔 접속

http://localhost:3000을 열어 회원가입한 뒤 둘러보세요.
