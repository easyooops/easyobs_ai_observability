OTLP/HTTP (모든 언어)

OTLP/HTTP 통합

EasyObs는 OpenTelemetry SDK가 제공되는 모든 언어(Java, Go, .NET, Rust, Ruby, PHP 등)의 표준 OTLP/HTTP 트레이스를 그대로 수신합니다.

엔드포인트

경로설명
/v1/traces표준 OTLP 엔드포인트. 모든 OTel SDK에서 사용
/otlp/v1/traces레거시 EasyObs 경로 (Python SDK가 사용)

인증

Authorization: Bearer <your-ingest-token>

토큰은 Setup > Organizations > Services에서 발급받을 수 있습니다.

cURL 예시

curl -X POST http://localhost:8787/v1/traces \
  -H "Authorization: Bearer eobs_your_token" \
  -H "Content-Type: application/json" \
  --data @sample_ingest.json
