API Reference

API Reference

EasyObs exposes a RESTful API built with FastAPI. Interactive documentation is available at /docs (Swagger UI).

Interactive API Explorer — Visit http://127.0.0.1:8787/docs for the live Swagger UI with try-it-out functionality. Open Swagger UI →

Authentication

All API requests require authentication via Bearer token:

Authorization: Bearer <token>

Two token types exist:

TypePurposeHow to Obtain
Session JWTWeb console / management APIPOST /v1/auth/signin
Ingest TokenOTLP trace ingestion from SDKsOrganizations > Services

Base URL

http://127.0.0.1:8787    (local development)
https://easyobs.your.co  (production behind reverse proxy)

CodeMeaning

Rate Limiting

Default limits per ingest token:

  • Trace ingest: 1000 spans/second
  • Management API: 100 requests/minute
