Installation

Installation Guide

Install the EasyObs server across various environments — local development, production, and air-gapped networks.

System Requirements

ComponentMinimumRecommended
Python3.113.12
Node.js20.x20.x LTS
RAM512 MB2 GB+
Disk1 GB10 GB+ (depends on trace volume)
OSWindows 10+, macOS 12+, Ubuntu 22.04+

Local Development Setup

1. Clone the Repository

git clone <repository-url>
cd easyobs_ai_observability

2. Create Environment File

cp .env.sample .env

3. Python Virtual Environment & Dependencies

# Using uv (recommended)
uv venv
uv pip install -e ".[dev,judges,pricing]"

# Or using pip
python -m venv .venv
source .venv/bin/activate        # Linux/macOS
.\.venv\Scripts\Activate.ps1     # Windows
pip install -e ".[dev,judges,pricing]"

4. Web Console Dependencies

cd apps/web
npm install
cd ../..

5. Start Development Servers

# Unified script (starts both API + Web)
./scripts/run-dev.sh             # Linux/macOS
.\scripts\run-dev.ps1            # Windows

Production Deployment

PostgreSQL Backend

For production use PostgreSQL instead of SQLite:

# .env
EASYOBS_DATABASE_URL=postgresql+asyncpg://user:password@host:5432/easyobs
⚠️ Warning: SQLite is intended for development/demo only. Use PostgreSQL for production workloads with concurrent access.

API Server (Production)

# Direct execution
uvicorn easyobs.http_app:app --host 0.0.0.0 --port 8787 --workers 4

# Or via environment variables
EASYOBS_API_HOST=0.0.0.0 EASYOBS_API_PORT=8787 easyobs serve

Web Console Build (Production)

cd apps/web
npm run build
npm start

Air-Gapped / Offline Installation

For environments without PyPI access:

1. Build Wheels (internet-connected machine)

.\scripts\build-agent-sdk.ps1 -IncludeDeps

2. Output Structure

dist/agent/
├─ easyobs_agent-0.1.0-py3-none-any.whl   ← SDK (~15KB)
└─ deps/                                    ← Dependencies
   ├─ opentelemetry_api-*.whl
   ├─ opentelemetry_sdk-*.whl
   ├─ opentelemetry_exporter_otlp_proto_http-*.whl
   └─ ... (transitive dependencies)

3. Install on Target

pip install --no-index --find-links ./deps ./easyobs_agent-0.1.0-py3-none-any.whl

Verify Installation

# API health check
curl http://127.0.0.1:8787/healthz
# Expected: {"status": "ok"}

# Backend tests
pytest -q

# Frontend build verification
cd apps/web && npm run build
