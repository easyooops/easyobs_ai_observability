LangChain Integration

Zero-config tracing for LangChain applications using the built-in EasyObsCallbackHandler.

Installation

pip install -e ".[langchain]"

This installs: opentelemetry-api, opentelemetry-sdk, opentelemetry-exporter-otlp-proto-http, and langchain-core.

Option A: Auto Mode (Recommended)

Simply set auto=True in init() — all LangChain invocations are captured globally without explicit callback wiring:

from easyobs_agent import init

init(
    "http://localhost:8787",
    token="eobs_...",
    service="my-langchain-app",
    auto=True,              # enables auto_langchain + capture_io
    session_id="sess_123",  # optional default session
    user_id="alice",        # optional default user
)

# That's it! Every llm.invoke(), chain.invoke(), retriever.invoke()
# is now automatically traced without any code changes.
💡 How it works: auto_langchain=True registers a process-global callback via langchain_core.tracers.context.register_configure_hook — the same mechanism LangSmith uses.

Option B: Explicit Callback

For more control, pass the handler explicitly:

from easyobs_agent import init
from easyobs_agent.callbacks import EasyObsCallbackHandler

init("http://localhost:8787", token="eobs_...", service="my-app")

handler = EasyObsCallbackHandler(session_id="sess_123", user_id="alice")

# Pass to individual invocations
result = llm.invoke("Hello", config={"callbacks": [handler]})
result = chain.invoke({"query": "..."}, config={"callbacks": [handler]})

Full Example

from easyobs_agent import init
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

# One-time setup
init("http://localhost:8787", token="eobs_...", service="rag-bot", auto=True)

# Build your chain normally — tracing is automatic
llm = ChatOpenAI(model="gpt-4o-mini")
prompt = ChatPromptTemplate.from_template("Answer: {question}")
chain = prompt | llm | StrOutputParser()

# Every invoke is automatically traced
answer = chain.invoke({"question": "What is EasyObs?"})
