Configuration

Configuration

All environment variables are optional. Defaults are defined in src/easyobs/settings.py.

How to set: Add to a .env file, or inject as real environment variables in your container/serverless runtime.

Storage

VariableDefaultDescription
EASYOBS_DATA_DIR./dataDirectory for SQLite catalog, JWT secret, and trace blobs. Wiped by easyobs reset-data.
EASYOBS_DATABASE_URL(auto)Leave blank → <DATA_DIR>/catalog.sqlite3. Production: postgresql+asyncpg://user:pw@host:5432/easyobs
EASYOBS_STORAGE_FORMATparquetTrace span storage format. parquet: columnar with DuckDB acceleration (recommended). ndjson: legacy JSON-lines.
EASYOBS_QUERY_ENGINEduckdbAnalytics query engine. duckdb: vectorized SQL over Parquet (scales to millions). legacy: Python loops (small deployments only).

HTTP Server

VariableDefaultDescription
EASYOBS_API_HOST127.0.0.1Bind address. Use 0.0.0.0 in containers.
EASYOBS_API_PORT8787API + OTLP/HTTP ingest port
EASYOBS_CORS_ORIGINShttp://127.0.0.1:3000,...Comma-separated browser origin whitelist

Authentication / Sessions

VariableDefaultDescription
EASYOBS_JWT_SECRET(auto-generated)HS256 signing key. Auto-created on first boot at <DATA_DIR>/jwt.secret. Production: use openssl rand -hex 32.
EASYOBS_JWT_TTL_HOURS12JWT expiration time in hours

LLM Cost Pricing

VariableDefaultDescription
EASYOBS_PRICING_SOURCEautoSource for o.price enrichment on ingest. auto: tokencost → litellm → builtin fallback.

Logging / Operational Visibility

VariableDefaultDescription
EASYOBS_LOG_LEVELINFOLog level: DEBUG / INFO / WARNING / ERROR
EASYOBS_LOG_FORMATconsoleconsole: human-readable single-line. json: CloudWatch/Loki compatible.
EASYOBS_LOG_FILE(none)If set, mirrors logs to this file in addition to stdout
EASYOBS_LOG_REQUEST_BODYfalseWhen true, logs request body (max 4KB). PII risk — off by default.

Demo / Mock Data

VariableDefaultDescription
EASYOBS_SEED_MOCK_DATAfalseWhen true, generates demo traces on first boot
EASYOBS_SEED_MOCK_TRACES100Number of synthetic traces to generate
EASYOBS_SEED_MOCK_WINDOW_HOURS24Time window for spreading synthetic traces
EASYOBS_SEED_MOCK_LIVEfalseContinuously drip synthetic traces while server runs
EASYOBS_SEED_MOCK_LIVE_INTERVAL_SEC20Interval between live synthetic traces (seconds)

Quality (Evaluation) Module

VariableDefaultDescription
EASYOBS_EVAL_ENABLEDtrueMaster switch. When false, all evaluation routes/UI are hidden.
EASYOBS_EVAL_AUTO_RULE_ON_INGESTtrueAuto-trigger rule evaluations on trace ingest

Alarms (Threshold Alerting)

VariableDefaultDescription
EASYOBS_ALARM_ENABLEDtrueMaster switch for the Alarms module
EASYOBS_ALARM_EVAL_INTERVAL_SECONDS60Alarm evaluation cadence (seconds)

Hybrid Storage (Local Blob + S3 Archive)

New in v0.2: Set EASYOBS_BLOB_PROVIDER=hybrid to enable dual-write to Local Parquet (hot, last 7 days) and S3 (cold archive, all data).

Query Routing

  • Preset windows (1h, 6h, 24h, 7d) → Local Blob Store (fast DuckDB scan)
  • Custom range beyond hot window → S3 Archive (DuckDB httpfs)
VariableDefaultDescription
EASYOBS_BLOB_PROVIDERlocalSet to hybrid for Local+S3 dual-write
EASYOBS_BLOB_BUCKET(none)S3 bucket name
EASYOBS_BLOB_PREFIXtracesObject key prefix
EASYOBS_BLOB_REGION(none)AWS region
EASYOBS_BLOB_S3_ACCESS_KEY_ID(none)AWS access key (optional if using IAM role)
EASYOBS_BLOB_S3_SECRET_ACCESS_KEY(none)AWS secret key
EASYOBS_BLOB_HOT_RETENTION_DAYS7Days of data in hot store; custom queries beyond this route to S3

Frontend (Web Console)

VariableDefaultDescription
NEXT_PUBLIC_API_URLhttp://127.0.0.1:8787API base URL the console calls

AWS Bedrock (Judge)

VariableDefaultDescription
AWS_PROFILEdefaultboto3 session profile (~/.aws/credentials)

Example .env File

# === Local Development ===
EASYOBS_DATA_DIR=./data
EASYOBS_API_HOST=127.0.0.1
EASYOBS_API_PORT=8787
EASYOBS_STORAGE_FORMAT=parquet
EASYOBS_QUERY_ENGINE=duckdb
EASYOBS_LOG_LEVEL=INFO
EASYOBS_LOG_FORMAT=console
EASYOBS_SEED_MOCK_DATA=true
EASYOBS_SEED_MOCK_TRACES=1000

# === Production (Hybrid: Local + S3) ===
# EASYOBS_DATABASE_URL=postgresql+asyncpg://user:pw@db:5432/easyobs
# EASYOBS_STORAGE_FORMAT=parquet
# EASYOBS_QUERY_ENGINE=duckdb
# EASYOBS_BLOB_PROVIDER=hybrid
# EASYOBS_BLOB_BUCKET=my-easyobs-archive
# EASYOBS_BLOB_PREFIX=traces
# EASYOBS_BLOB_REGION=ap-northeast-2
# EASYOBS_BLOB_HOT_RETENTION_DAYS=7
# EASYOBS_API_HOST=0.0.0.0
# EASYOBS_JWT_SECRET=your-strong-secret-here
